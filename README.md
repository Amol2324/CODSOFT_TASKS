# CodSoft Machine Learning Internship Tasks

This repository contains all three projects completed during the CodSoft Machine Learning Internship.

## Tasks

### Task 1 - Movie Genre Classification
- Predict movie genres from plot summaries
- Algorithms:
  - TF-IDF
  - Logistic Regression

Folder: Task_1_MOVIE_GENRE


### Task 2 - Customer Churn Prediction
- Predict whether a customer will leave a telecom company
- Algorithms:
  - Logistic Regression
  - Random Forest
  - Gradient Boosting

Folder: Task_2_CHURN_PREDICTION

### Task 3 - SMS Spam Detection
- Classify SMS messages as Spam or Ham
- Algorithms: TF-IDF + Logistic Regression

Folder: Task_1_SPAM_SMS_DETECTION
